Sistema de Administración de Mandatarios de la Provincia de Santa Fe

Para utilizar el sistema debe ingresar el usuario y la clave que le fue provista para tal fin. No compartas esta información con ninguna persona que no sea la autorizada para ingresar al sistema.

Identifícate para utilizar el sistema

Ingresa tu usuario y clave y luego presiona el botón Ingresar

  • Tené en cuenta que tanto el usuario y la clave son sensitivas a mayúsculas y minúsculas
  • Recordá cerrar la sesión cuando termines de utilizar el sistema
